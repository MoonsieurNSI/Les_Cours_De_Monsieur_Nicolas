# Mise à jour Depuis sa Console
- `git pull` pour mettre à jour son repo local avec le repo sur github

# En console, après modifications locales
- `git status` pour check
- `git add ./` pour ajouter tous les changements du dossier courant
- `git commit -m "mon message"` pour la sauvegarde locale
- `git push` pour envoyer les modifs sur github
